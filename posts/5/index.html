在Dreamhost上使用svnmailer

Jan 5th, 2010 12:00 am | Comments

首先你要装好SVN,可以参考我以前的文章: http://weixuhong.com/%E6%8A%80%E6%9C%AF%E6%84%9F%E6%83%B3/2008/09/27/%E5%9C%A8dreamhost%E4%B8%8A%E9%85%8D%E7%BD%AEtrac-and-svn/

svnmailer官方网站: http://opensource.perlig.de

svnmailer的作用就是当开发者，提交代码的时候，会给团队人员发送一份email，内容包括修改的文件及内容.

1. 先下载一个svnmailer的版本

wget http://storage.perlig.de/svnmailer/svnmailer-1.0.8.tar.gz

2. 安装svnmailer

python setup.py install

3. 生成svnmailer的配置文件

可在任何位置生成这个配置文件，这里我们在/home/weston/svn/svnmailer.conf,内容如下:

[general]
smtp_host = localhost:25
config_charset = utf-8

[author table] weston = youremail@gmail.com

[maps] from_addr = [author table] to_addr = [author table]

[defaults] commit_subject_prefix = [svn] commit: apply_charset_property = yes show_applied_charset = yes from_addr = %(author)s to_addr = weston browser_base_url = Trac http://trac.weixuhong.com/red-cn.cgi/browser mail_transfer_encoding = 8bit

* 其中smtp_host = localhost:25是你发送email的smtp服务器
* weston = youremail@gmail.com 是谁接收email

4. 配置项目，让svnmailer工作

A. 首先到你的项目的hook文件夹下面，然后复制post-commit.tmpl 为post-commit，

cp post-commit.tmpl post-commit

如果使用SSH隧道上网–For Mac篇

Nov 14th, 2009 12:00 am | Comments

本教程限使用在Linux或者Mac下面，但是如果在Windows使用，原理一样(但是需要用Putty登录代理主机), 本教程或许需要你有一定的计算机知识，一些设置也可以在视频找到。

  1. 先要下载安装FireFox浏览器
    下载地址: www.mozilla.com/firefox/

  2. 安装FireFox插件：FoxyProxy

  3. 安装好FireFox之后，请在插件处，查找安装FoxyProxy, 如果不知道如何安装插件，请用Google搜索或者咨询我.

  4. 配置FoxyProxy
    详细的配置在视频当中,请参考如何设置即可.

  5. 使用Shell用SSH命令登录指定帐号（如果没有帐号，请咨询我，我会适当的发一些帐号)
    在Shell下面使用以下命令
    ssh -N -D 9999 test@example.com
    test: 为用户名
    example.com： 为您有登录权限的国外主机

*Windows用户请用Putty设置登录主机.
